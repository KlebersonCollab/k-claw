# Coding Standards & Quality

- **Naming**: Use snake_case for Python and camelCase for TypeScript.
- **Async**: Always use `async/await` for I/O operations; never use `time.sleep()`.
- **Typing**: All functions must have type hints (Python) or explicit interfaces (TS).
- **Tests**: No code is complete without its corresponding test file in `tests/`.
- **Imports**: Organize imports alphabetically and group by (standard, third-party, local).
